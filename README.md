##### # 08/04

보통 SI 프로젝트에선 SVN을 많이 사용하는데,

개인 프로젝트나 자사 서비스를 운영하는 곳에선 대부분 Git으로 코드관리를 한다.

Git은 DVCS로 **코드 관리를 로컬저장소와 원격저장소 양쪽에서 관리**할 수 있는 장점.

(SVN은 원격저장소밖에 없다.)

굳이 코드 관리용 서버를 구축할 필요 없이 Github를 사용하면 바로 원격 저장소를 사용 가능.

Git + Source Tree를 사용.
